# 网文平台自动发布系统使用指南

## 系统概述

这是一个专为《万界神级系统》设计的多平台自动发布系统，支持起点中文网、飞卢小说网、17K小说网等主流网文平台的章节自动发布、审核状态监控和数据统计。

## 快速开始

### 1. 安装依赖
```bash
pip install requests beautifulsoup4 lxml
```

### 2. 配置平台信息
编辑 `platform_config.json` 文件，填入各平台的API密钥和作品信息：

```json
{
  "platforms": {
    "qidian": {
      "enabled": true,
      "config": {
        "token": "你的起点API密钥",
        "book_id": "作品ID"
      }
    }
  }
}
```

### 3. 发布章节
```python
from auto_publisher import PlatformPublisher, ChapterData

# 创建发布器
publisher = PlatformPublisher()

# 创建章节数据
chapter = ChapterData(
    title="第一章：系统觉醒",
    content="章节内容...",
    chapter_number=1,
    word_count=3500
)

# 发布到所有启用的平台
results = publisher.publish_chapter(chapter)
print(results)
```

- **优势**：审核宽松，支持批量导入

- **分成比例**：60%

### 17K小说网
- **优势**：支持Markdown，社区活跃
- **要求**：实名认证，1800字/章起
- **审核时间**：1-12小时
- **分成比例**：55%

### 1. 定时发布

import schedule
import time

def daily_publish():
    # 每日定时发布
    publisher = PlatformPublisher()
    # 获取待发布章节
    chapter = get_next_chapter()

    time.sleep(60)

## 故障排除

### 常见问题

#### 1. API连接失败
```
错误：ConnectionError
解决：检查网络连接和API密钥
```

#### 2. 审核被拒绝
```
错误：Chapter rejected
解决：检查敏感词，调整内容
```

#### 3. 发布频率超限
```
错误：Rate limit exceeded
解决：降低发布频率，等待重置
```
